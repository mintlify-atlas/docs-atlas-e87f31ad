## useComprehensiveWallet

Comprehensive wallet hook using Dynamic SDK with multi-wallet support, balance tracking, and transaction management.

### Import

```typescript
import { useComprehensiveWallet } from '@/hooks/useComprehensiveWallet';
```

### Usage

```typescript
function MyComponent() {
  const {
    connected,
    address,
    balances,
    connect,
    disconnect,
    sendTransaction,
    sendToken,
    refreshBalances,
  } = useComprehensiveWallet();

  return (
    <div>
      {connected ? (
        <>
          <p>Address: {address}</p>
          <p>ETH Balance: {balances.eth}</p>
          <p>USDC Balance: {balances.usdc}</p>
          <button onClick={disconnect}>Disconnect</button>
        </>
      ) : (
        <button onClick={connect}>Connect Wallet</button>
      )}
    </div>
  );
}
```

  - `BTC`: `0xD8a6E3FCA403d79b6AD6216b60527F51cc967D39`
</ResponseField>

<ResponseField name="tokens" type="object">
  Token configurations with decimals and metadata
</ResponseField>

### Methods

#### connect()

Open Dynamic auth flow to connect wallet.

```typescript
const { connect } = useComprehensiveWallet();

await connect();
```

#### disconnect()

Disconnect wallet and clear balances.

```typescript
const { disconnect } = useComprehensiveWallet();

await disconnect();
```

#### sendTransaction()

Send ETH transaction.

<ParamField path="params" type="TransactionParams" required>
  Transaction parameters
</ParamField>

<ParamField path="params.to" type="string" required>
  Recipient address
</ParamField>

<ParamField path="params.value" type="string">
  Amount in ETH (e.g., "0.1")
</ParamField>

<ParamField path="params.data" type="string">
  Transaction data (hex)
</ParamField>

<ParamField path="params.gasLimit" type="string">
  Gas limit (hex)
</ParamField>

**Returns:** `Promise<{ hash: string }>`

```typescript
const { sendTransaction } = useComprehensiveWallet();

const result = await sendTransaction({
  to: '0x742d35Cc6634C0532925a3b844Bc9e7595f0bEb',
  value: '0.1', // 0.1 ETH
});

console.log('Transaction hash:', result.hash);
```

#### sendToken()

Send ERC20 tokens.

<ParamField path="params" type="SendTokenParams" required>
  Token transfer parameters
</ParamField>

<ParamField path="params.tokenAddress" type="string" required>
  Token contract address
</ParamField>

<ParamField path="params.to" type="string" required>
  Recipient address
</ParamField>

<ParamField path="params.amount" type="string" required>
  Amount to send (in token units)
</ParamField>

<ParamField path="params.decimals" type="number">
  Token decimals (defaults to 18)
</ParamField>

**Returns:** `Promise<{ hash: string }>`

```typescript
const { sendToken, tokenAddresses } = useComprehensiveWallet();

const result = await sendToken({
  tokenAddress: tokenAddresses.USDC,
  to: '0x742d35Cc6634C0532925a3b844Bc9e7595f0bEb',
  amount: '100',
  decimals: 6,
});
```

#### signMessage()

Sign a message with the wallet.

<ParamField path="message" type="string" required>
  Message to sign
</ParamField>

**Returns:** `Promise<string>` - Signature

```typescript
const { signMessage } = useComprehensiveWallet();

const signature = await signMessage('Hello, Web3!');
console.log('Signature:', signature);
```

#### refreshBalances()

Manually refresh all token balances.

```typescript
const { refreshBalances } = useComprehensiveWallet();

await refreshBalances();
```

<Note>
  Balances are automatically refreshed every 10 seconds when wallet is connected.
</Note>

#### Multi-Wallet Methods

<ResponseField name="addWallet" type="() => Promise<void>">
  Open Dynamic's add wallet flow
</ResponseField>

<ResponseField name="openAuthFlow" type="() => void">
  Manually open Dynamic authentication modal
</ResponseField>

## useAgwWallet

### Import

### Usage

```typescript
function DepositPage() {
  const {
    connected,
    address,
    ethBalance,
    usdcBalance,
    btcBalance,
    signIn,
    signOut,
    sendTransaction,
    sendToken,
    refreshBalance,

  if (!connected) {
```

### Methods

#### signIn()

Open Dynamic login modal to create embedded wallet.

```typescript
const { signIn } = useAgwWallet();

await signIn();
```

#### signOut() / disconnect()

Logout and clear wallet data.

```typescript
const { signOut } = useAgwWallet();

await signOut();
```

<Note>
  `disconnect()` is an alias for `signOut()`.
</Note>

#### refreshBalance()

Refresh all token balances (ETH + 10 tokens).

```typescript
const { refreshBalance } = useAgwWallet();

await refreshBalance();
```

#### getAddress()

Get current wallet address.

**Returns:** `string | null`

```typescript
const { getAddress } = useAgwWallet();

const address = getAddress();
```

#### sendTransaction()

Send ETH transaction with optional contract data.

<ParamField path="to" type="string" required>
  Recipient address
</ParamField>

<ParamField path="value" type="string" required>
  Amount in ETH
</ParamField>

<ParamField path="tokenType" type="'ETH' | 'USDC'">
  Token type (defaults to 'ETH')
</ParamField>

<ParamField path="data" type="string">
  Contract call data (hex)
</ParamField>

<ParamField path="skipSave" type="boolean">
  Skip saving transaction to localStorage
</ParamField>

**Returns:** `Promise<{ hash: string, receipt: TransactionReceipt }>`

```typescript
const { sendTransaction } = useAgwWallet();

const result = await sendTransaction(
  '0x742d35Cc6634C0532925a3b844Bc9e7595f0bEb',
  '0.05', // 0.05 ETH
  'ETH'
);

console.log('TX Hash:', result.hash);
console.log('Receipt:', result.receipt);
```

#### sendToken()

Send ERC20 tokens with automatic transaction encoding.

<ParamField path="tokenAddress" type="string" required>
  Token contract address
</ParamField>

<ParamField path="to" type="string" required>
  Recipient address
</ParamField>

<ParamField path="amount" type="string" required>
  Amount to send
</ParamField>

<ParamField path="decimals" type="number">
  Token decimals (defaults to 6)
</ParamField>

**Returns:** `Promise<{ hash: string, receipt: TransactionReceipt }>`

```typescript
const { sendToken, USDC_ADDRESS } = useAgwWallet();

const result = await sendToken(
  USDC_ADDRESS,
  '0x742d35Cc6634C0532925a3b844Bc9e7595f0bEb',
  '100', // 100 USDC
  6
);
```

#### addPasskey()

Open Dynamic user profile to add passkey MFA.

```typescript
const { addPasskey } = useAgwWallet();

await addPasskey();
```

<Warning>
  Requires wallet to be connected. Shows warning toast if not connected.
</Warning>

#### exportPrivateKey()

Open Dynamic settings to export private key.

```typescript
const { exportPrivateKey } = useAgwWallet();

await exportPrivateKey();
```

<Warning>
  Requires MFA/Passkey setup in Dynamic dashboard. Users must add passkey before exporting.
</Warning>

### Performance Optimizations

<AccordionGroup>
  <Accordion title="Wallet Client Caching">
    The hook caches the wallet client to avoid repeated async calls:
    
    ```typescript
    // Wallet client is preloaded when wallet connects
    useEffect(() => {
      if (primaryWallet) {
        primaryWallet.getWalletClient?.().then((client) => {
          walletClientRef.current = client;
        });
      }
    }, [primaryWallet]);
    ```
  </Accordion>

  <Accordion title="Parallel Balance Fetching">
    All token balances are fetched in parallel with Promise.allSettled:
    
    ```typescript
    const [ethBal, usdcData, btcData, ...] = await Promise.allSettled([
      fetchEthBalance(),
      fetchTokenBalance(USDC_ADDRESS),
      fetchTokenBalance(BTC_ADDRESS),
      // ... all other tokens
    ]);
    ```
  </Accordion>

  <Accordion title="Request Timeout Protection">
    10-second timeout prevents hanging requests:
    
    ```typescript
    const controller = new AbortController();
    const timeoutId = setTimeout(() => controller.abort(), 10000);
    ```
  </Accordion>
</AccordionGroup>

## useDynamicWallet

Lightweight wallet hook for basic wallet operations.

### Import

```typescript
import { useDynamicWallet } from '@/hooks/usePrivyWallet';
```

### Usage

```typescript
function SimpleWallet() {
  const {
    isConnected,
    address,
    connect,
    disconnect,
    getBalance,
    send,
  } = useDynamicWallet();

  return (
    <div>
      {isConnected ? (
        <>
          <p>{address}</p>
          <button onClick={disconnect}>Disconnect</button>
        </>
      ) : (
        <button onClick={connect}>Connect</button>
      )}
    </div>
  );
}
```

### Return Values

<ResponseField name="isReady" type="boolean">
  Always `true` (hook is ready)
</ResponseField>

<ResponseField name="isConnected" type="boolean">
  Whether wallet is connected
</ResponseField>

<ResponseField name="address" type="string | undefined">
  Connected wallet address
</ResponseField>

<ResponseField name="user" type="User | null">
  User profile from Dynamic
</ResponseField>

<ResponseField name="chainId" type="number">
  Base Sepolia chain ID (84532)
</ResponseField>

<ResponseField name="chainName" type="string">
  "Base Sepolia"
</ResponseField>

<ResponseField name="primaryWallet" type="Wallet | null">
  Primary wallet object
</ResponseField>

### Methods

#### connect()

```typescript
const { connect } = useDynamicWallet();
await connect();
```

#### disconnect()

Disconnects with 2-second timeout for faster UX.

```typescript
const { disconnect } = useDynamicWallet();
await disconnect();
```

#### getBalance()

Fetch ETH balance.

**Returns:** `Promise<string>`

```typescript
const { getBalance } = useDynamicWallet();

const balance = await getBalance();
console.log('Balance:', balance, 'ETH');
```

#### send()

Send ETH transaction.

<ParamField path="to" type="string" required>
  Recipient address
</ParamField>

<ParamField path="amount" type="string" required>
  Amount in ETH
</ParamField>

**Returns:** `Promise<{ hash: string }>`

```typescript
const { send } = useDynamicWallet();

const result = await send(
  '0x742d35Cc6634C0532925a3b844Bc9e7595f0bEb',
  '0.01'
);
```

## Hook Comparison

<Tabs>
  <Tab title="useComprehensiveWallet">
    **Best for:** Full-featured applications requiring multi-wallet support

    **Features:**
    - Multi-wallet management
    - Auto-refreshing balances (10s interval)
    - Token balance tracking (ETH, USDC, BTC)
    - Message signing
    - Network switching
    - Comprehensive error handling

    **Use cases:**
    - DeFi dashboards
    - Portfolio trackers
    - Multi-chain applications
  </Tab>

  <Tab title="useAgwWallet">
    **Best for:** Production apps with 10+ token support

    **Features:**
    - 10+ token balances (BTC, ETH, SOL, BNB, etc.)
    - Wallet client caching
    - Parallel balance fetching
    - Transaction history
    - Passkey MFA support
    - Private key export

    **Use cases:**
    - Token trading platforms
    - Multi-token wallets
    - Deposit/withdrawal systems
  </Tab>

  <Tab title="useDynamicWallet">
    **Best for:** Simple applications with basic wallet needs

    **Features:**
    - Lightweight (~100 LOC)
    - Basic ETH operations
    - Fast disconnect (2s timeout)
    - Minimal dependencies

    **Use cases:**
    - Simple dApps
    - Authentication-only apps
    - Prototype applications
  </Tab>
</Tabs>

## Common Patterns

### Send USDC Transaction

```typescript
const { sendToken, tokenAddresses } = useComprehensiveWallet();

const result = await sendToken({
  tokenAddress: tokenAddresses.USDC,
  to: recipientAddress,
  amount: '100',
  decimals: 6,
});
```

### Check Multiple Balances

```typescript
const { getAllTokenBalances } = useComprehensiveWallet();

const balances = getAllTokenBalances();
balances.forEach(token => {
  console.log(`${token.symbol}: ${token.balance}`);
});
```

### Handle Transaction Errors

```typescript
const { sendTransaction } = useAgwWallet();

try {
  const result = await sendTransaction(to, amount, 'ETH');
  console.log('Success:', result.hash);
} catch (error) {
  // Error automatically shown via toast
  console.error('Transaction failed:', error);
}
```

### Multi-Wallet Support

```typescript
const { userWallets, switchWallet, addWallet } = useComprehensiveWallet();

// List all wallets
userWallets.forEach(wallet => {
  console.log(wallet.address);
});

// Switch to different wallet
await switchWallet(walletId);

// Add new wallet
await addWallet();
```
