## Overview

GweAI uses [Wagmi](https://wagmi.sh) for Ethereum wallet connections and blockchain interactions. The configuration includes automatic RPC fallback, multiple wallet connectors, and retry logic for reliable connectivity.

## Configuration

The Wagmi configuration is exported from `wagmi.config.ts`:

```typescript wagmi.config.ts
import { http, createConfig, fallback } from 'wagmi';
import { baseSepolia } from 'wagmi/chains';
import { injected, coinbaseWallet, walletConnect } from 'wagmi/connectors';

const BASE_SEPOLIA_RPCS = [
  'https://base-sepolia.g.alchemy.com/v2/-mGklZw8tTiO9fg9sRGQP',
  'https://base-sepolia.blockpi.network/v1/rpc/public',
  'https://base-sepolia-rpc.publicnode.com',
];

export const wagmiConfig = createConfig({
  chains: [baseSepolia],
  connectors: [
    injected(),
    coinbaseWallet({
      appName: 'GweAI',
      preference: 'smartWalletOnly',
    }),
    walletConnect({
      projectId: '3fbb6bba6f1de962d911bb5b5c9ddd26',
      metadata: {
        name: 'GweAI',
        description: 'AI-Powered DeFi Trading Platform',
        url: 'https://gweai.com',
        icons: ['https://gweai.com/icon.png'],
      },
      showQrModal: true,
    }),
  ],
  transports: {
    [baseSepolia.id]: fallback(
      BASE_SEPOLIA_RPCS.map(url => http(url, {
        timeout: 10000,
        retryCount: 3,
        retryDelay: 150,
      }))
    ),
  },
});
```

## Wallet Connectors

### Injected Connector

Supports browser extension wallets:

<CodeGroup>
```typescript Usage
import { injected } from 'wagmi/connectors';

const connector = injected();
```
</CodeGroup>

**Supported Wallets:**
- MetaMask
- Brave Wallet
- Rabby Wallet
- Any EIP-1193 compatible wallet

### Coinbase Wallet

Configured for Coinbase Smart Wallet (embedded wallet experience):

<ParamField path="appName" type="string" required>
  Application name displayed in Coinbase Wallet (set to `"GweAI"`)
</ParamField>

<ParamField path="preference" type="string" default="smartWalletOnly">
  Wallet type preference. Set to `smartWalletOnly` for embedded wallet experience
</ParamField>

<CodeGroup>
```typescript Configuration
import { coinbaseWallet } from 'wagmi/connectors';

const connector = coinbaseWallet({
  appName: 'GweAI',
  preference: 'smartWalletOnly',
});
```
</CodeGroup>

## Transport Configuration

### Fallback Transport

The configuration uses Wagmi's `fallback` transport for automatic RPC failover:

<ParamField path="timeout" type="number" default="10000">
  Request timeout in milliseconds (10 seconds)
</ParamField>

<ParamField path="retryCount" type="number" default="3">
  Number of retry attempts before moving to next RPC
</ParamField>

<ParamField path="retryDelay" type="number" default="150">
  Delay between retry attempts in milliseconds
</ParamField>

### RPC Endpoints

The transport uses multiple RPC providers in priority order:

<Steps>
  <Step title="Alchemy (Primary)">
    `https://base-sepolia.g.alchemy.com/v2/-mGklZw8tTiO9fg9sRGQP`
    
    Best rate limits and reliability
  </Step>
  
  <Step title="BlockPI (Secondary)">
    `https://base-sepolia.blockpi.network/v1/rpc/public`
    
    Public endpoint with 10M requests/day
  </Step>
  
  <Step title="PublicNode (Tertiary)">
    `https://base-sepolia-rpc.publicnode.com`
    
    Community-run fallback endpoint
  </Step>
</Steps>

<Note>
  **Base Sepolia's official RPC** (`https://sepolia.base.org`) was removed due to aggressive rate limiting causing 403 errors. The fallback strategy provides 99.9% uptime.
</Note>

## Usage Examples

### Initialize Wagmi Config

<CodeGroup>
```typescript App Setup
import { WagmiProvider } from 'wagmi';
import { QueryClient, QueryClientProvider } from '@tanstack/react-query';
import { wagmiConfig } from './wagmi.config';

const queryClient = new QueryClient();

function App() {
  return (
    <WagmiProvider config={wagmiConfig}>
      <QueryClientProvider client={queryClient}>
        {/* Your app components */}
      </QueryClientProvider>
    </WagmiProvider>
  );
}
```

```typescript Connect Wallet
import { useConnect } from 'wagmi';

function ConnectButton() {
  const { connectors, connect } = useConnect();

  return (
    <div>
      {connectors.map((connector) => (
        <button
          key={connector.id}
          onClick={() => connect({ connector })}
        >
          Connect {connector.name}
        </button>
      ))}
    </div>
  );
}
```

```typescript Read Blockchain Data
import { useAccount, useBalance } from 'wagmi';

function WalletInfo() {
  const { address } = useAccount();
  const { data: balance } = useBalance({ address });

  return (
    <div>
      <p>Address: {address}</p>
      <p>Balance: {balance?.formatted} ETH</p>
    </div>
  );
}
```
</CodeGroup>
